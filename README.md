## 🇬🇧 English

### 📖 About the Project

This project predicts whether a patient has diabetes using Machine Learning.

The model is trained with the **Pima Indians Diabetes Dataset** and uses the **Logistic Regression** algorithm from Scikit-learn.

The dataset contains medical information such as:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

The model predicts whether a patient is diabetic (`1`) or not (`0`).
